# Floral

## How to start?

##BackEnd

#cd server
```
npm install
```

```
node server.js
```
##FrontEnd

#cd client
```
npm install
```

```
npm run dev
```

## How to run?

Open http://localhost:3000 with your browser to see the result.

## Used technologies:

- Frontend: React Js -> Axios, React Router v-6, SCSS, Ant Design, Redux, Redux-thunk, embla-carousel
- BackEnd: mongodb, express, node JS, 
- Git: GitHub
